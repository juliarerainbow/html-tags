HTML TAGS

Per creare questo documento si è chiesto il supporto di chatbot, delle risorse da Mozilla Developer Network e da w3schools e probabilmente altro

Lista dei tag

  • Tag: < a >
  • Utilizzo : crea un collegamento ipertestuale in un documento HTML. Serve a definire un link che permette agli utenti di navigare verso un'altra pagina web, un'altra sezione dello stesso documento o eseguire altre azioni specificate tramite l'attributo href
  • Esempio :
    sito esempio
  • Tag: < abbr >
  • Utilizzo : viene utilizzato per segnalare e definire un'abbreviazione o un acronimo presente nel testo. Fornisce un'espansione completa o una descrizione dell'abbreviazione, che può essere visualizzata come un tooltip o un'indicazione visiva quando l'utente interagisce con l'abbreviazione nel browser.
  • Esempio :
    Il WWW è stato introdotto nel 1989.
  • Tag: < address >
  • Utilizzo : viene utilizzato in una frase per definire le informazioni di contatto o l'indirizzo di un'autore o di una sezione di contatto all'interno di un documento HTML.
  • Esempio :
    Ste Ghe
    Via Conosciuta 3 int 5
    Genova, Italia
    Email: sonounamaildiesempio@nonsaprei.com
    Cel: 123-456-7890
  • Tag: < area >
  • Utilizzo : in una frase viene utilizzato per definire un'area interattiva all'interno di una mappa di immagine (tag < map >) in un documento HTML. Serve a specificare un'area cliccabile o selezionabile all'interno di un'immagine, che può essere associata a un collegamento ipertestuale o a un'azione specifica.
  • Esempio :
  • Tag: < article >
  • Utilizzo : viene utilizzato in una frase per definire un contenuto autonomo , indipendente e significativo all'interno di un documento HTML. Serve a identificare una sezione o un blocco di contenuto che costituisce un' entità completa e che può essere distribuita, ripubblicata o riutilizzata separatamente da altre parti del documento o in contesti diversi.
  • Esempio :

    Weather forecast for Seattle

    03 March 2018

    Rain.

    04 March 2018

    Periods of rain.

    05 March 2018

    Heavy rain.

  • Tag: < aside >
  • Utilizzo : viene utilizzato per definire un contenuto ausiliario o correlato all'elemento principale di una pagina o di una sezione. Serve a fornire informazioni aggiuntive, contenuti relativi, sidebar o elementi di supporto che sono separati dal flusso principale del documento.
  • Esempio :

    Ulteriori risorse

    • Link 1
    • Link 2
    • Link 3

    Sono un Header

    Sono un lorem ipsum dolor sit amet consectetur adipisicing elit. Perferendis, sit, beatae ut assumenda, aspernatur itaque quisquam accusamus placeat reiciendis earum dignissimos. Minus et iste dolor, animi quia voluptatibus debitis mollitia?


  • Tag: < audio >
  • Utilizzo : viene utilizzato per incorporare file audio all'interno di una pagina web. Serve a riprodurre contenuti audio, come brani musicali, effetti sonori o registrazioni vocali, direttamente nel browser dei visitatori del sito.
  • Esempio :
  • Tag: < b >
  • Utilizzo : viene utilizzato per applicare una formattazione di testo in grassetto (bold) al suo contenuto all'interno di un documento HTML. Serve a evidenziare visivamente una parola o una porzione di testo importante, senza apportare alcun significato semantico specifico .
  • Esempio :

    Sono un testo in grassetto.

  • Tag: < base >
  • Utilizzo : viene utilizzato per specificare l'URL di base o l'URL di base di destinazione per tutti i collegamenti relativi all'interno di un documento HTML. Serve a fornire un punto di riferimento comune per le risorse collegate, come file CSS, immagini o pagine web, consentendo di specificare URL relativi in modo più conciso.
  • Esempio : 
            <head>
            <base href="https://www.example.com/">
        </head>
  • Tag: < bdi > ??what???
  • Utilizzo :(Bidirectional Isolate) in una frase viene utilizzato per isolare una porzione di testo che ha una direzione di scrittura diversa dal testo circostante all'interno di un documento HTML. Serve a gestire correttamente la visualizzazione e la lettura di testo bidirezionale, come l'arabo o l'ebraico, in contesti multilingue.
  • Esempio :

    Alice,i topi non aveva non ipot i

  • Tag: < bdo >
  • Utilizzo :(Bidirectional Override) viene utilizzato per annullare la direzione di scrittura predefinita di un testo all'interno di un documento HTML. Con questo tag, è possibile forzare una direzione di scrittura specifica su un testo, indipendentemente dalla direzione di scrittura del resto del documento.
  • Esempio :

    Alice, i topi non avevano nipoti.

    Alice, i topi non avevano nipoti.

  • Tag: < blockquote >
  • Utilizzo : viene utilizzato per rappresentare un blocco di citazione o estratto di testo che è stato estratto da un'altra fonte o attribuito a una persona specifica. Il testo all'interno di esso viene di solito visualizzato con un rientro aggiuntivo rispetto al resto del contenuto della pagina, per indicare che si tratta di una citazione o di un estratto distinto.
  • Esempio :

    ‟ Due cose sono infinite: l’universo e la stupidità umana, ma riguardo l’universo ho ancora dei dubbi. ”

    —   Albert Einstein
  • Tag: < body >
  • Utilizzo : rappresenta il corpo principale di un documento HTML. Contiene tutto il contenuto visibile della pagina web, come testo, immagini, link, tabelle, video, script e altro ancora. E' uno dei tag fondamentali di un documento HTML e viene utilizzato per definire il contenuto visibile e interattivo che gli utenti vedranno nel browser. Tutto ciò che viene inserito all'interno del tag sarà visualizzato nel corpo della pagina web.
  • Esempio : 
            <body>
            <h1> Hello World</h1>
        </body>
  • Tag: < br >
  • Utilizzo : viene utilizzato per inserire un'interruzione di linea (line break) all'interno di un documento HTML. Non richiede un tag di chiusura perché è un elemento vuoto. Quando il browser incontra il tag, passa alla riga successiva, creando una nuova riga di testo. È utile per inserire spazi vuoti verticali tra i paragrafi, creare elenchi a capo, formattare l'indirizzario o in qualsiasi altra situazione in cui si desideri forzare un'interruzione di linea.
  • Esempio :

    Hello World,
    It's Me!

  • Tag: < button >
  • Utilizzo : viene utilizzato per creare un pulsante interattivo all'interno di un documento HTML. Questo elemento fornisce un modo per gli utenti di interagire con il contenuto della pagina web, come inviare un modulo, attivare una funzione JavaScript o eseguire un'azione specifica.
  • Esempio :

  • Tag: < canvas >
  • Utilizzo : viene utilizzato per creare grafica dinamica e disegnare elementi grafici, come immagini, forme, grafici e animazioni, all'interno di una pagina web utilizzando JavaScript. Il tag fornisce uno spazio bidimensionale o tridimensionale sul quale è possibile disegnare attraverso l'utilizzo delle API grafiche di HTML5 e JavaScript. È possibile controllare i pixel all'interno del canvas per creare grafica personalizzata, effetti visivi interattivi o giochi.
  • Esempio :
  • Tag: < caption >
  • Utilizzo : viene utilizzato per fornire un titolo o una descrizione per un elemento tabella (table) all'interno di un documento HTML. Il contenuto del tag viene posizionato di solito sopra o sotto la tabella per identificarne il contenuto o scopo.
  • Esempio :

    Caption : Tabella degli studenti iscritti
    ID Nome Cognome Anno di nascita
    001 Marco Rossi 1995
    002 Giovanna Bianchi 1998
  • Tag: < cite >
  • Utilizzo : viene utilizzato per indicare il titolo o la fonte di un'opera citata, come un libro, un articolo, una pubblicazione o una citazione all'interno di un documento HTML. Il contenuto del tag viene di solito visualizzato in corsivo per distinguere la citazione dal resto del testo. Può anche essere utilizzato per identificare una citazione all'interno di un blocco di testo o attribuire la citazione all'autore.Il tag fornisce un modo per evidenziare e rendere semanticamente corretta un'opera citata o una citazione all'interno di un documento HTML.
  • Esempio :
    ‟ Due cose sono infinite: l’universo e la stupidità umana, ma riguardo l’universo ho ancora dei dubbi. ”   Albert Einstein
  • Tag: < code >
  • Utilizzo : viene utilizzato per indicare il codice o frammenti di codice all'interno di un documento HTML. Il testo all'interno del tag viene di solito visualizzato con una formattazione a larghezza fissa (monospazio) per distinguere il codice dal resto del testo.
  • Esempio :

    Per stampare un messaggio in JavaScript, puoi utilizzare il seguente codice: console.log('Hello, world!');

  • Tag: < col > ??what???
  • Utilizzo : viene utilizzato per definire le proprietà di formattazione per una o più colonne all'interno di una tabella (table). Il tag viene solitamente utilizzato insieme al tag "colgroup" per raggruppare le colonne e applicare proprietà comuni a esse.
  • Esempio 1 :
    Colonna 1 Colonna 2
    Dato 1 Dato 2
    Esempio 2 :
    Frutta Quantità Prezzo
    Mela 5 1.50
    Pera 3 0.80
  • Tag: < colgroup > ??what???
  • Utilizzo : viene utilizzato per raggruppare uno o più tag "col" all'interno di una tabella ("table") per applicare proprietà comuni alle colonne. Questo permette di specificare le proprietà di formattazione delle colonne in modo più efficiente rispetto all'utilizzo separato dei tag "col".
  • Esempio :
    Colonna 1 Colonna 2 Colonna 3
    Dato 1 Dato 2 Dato 3
